# Bughub

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bughub', github: 'activebridge/bughub-ruby'
```

And then execute:

    $ bundle

## Usage

    $ rails g bughub [API-KEY]

OR

    $ rails g bughub
    $ export BUGHUB-API_KEY=[API_KEY]
