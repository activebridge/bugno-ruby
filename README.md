# Radar

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'radar', github: 'activebridge/radar-ruby'
```

And then execute:

    $ bundle

## Usage

    $ rails g radar [API-KEY]

OR

    $ rails g radar
    $ export RADAR-API_KEY=[API_KEY]
